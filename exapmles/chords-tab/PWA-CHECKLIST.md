# Чеклист для PWA установки на Android

## Требования для автоматического баннера установки в Chrome на Android:

1. ✅ **HTTPS** - сайт должен быть доступен по HTTPS (deepark.ru должен иметь SSL сертификат)
2. ✅ **Валидный manifest.json** - файл должен быть доступен и валиден
3. ✅ **Service Worker** - должен быть зарегистрирован и работать
4. ✅ **Иконки** - минимум 192x192 пикселей (icon-192.png и icon-512.png)
5. ✅ **Пользователь должен провести время на сайте** - минимум несколько секунд

## Проверка настройки:

### 1. Проверьте доступность файлов:
- `https://deepark.ru/chtab/index.html` - должен открываться
- `https://deepark.ru/chtab/manifest.json` - должен открываться и показывать JSON
- `https://deepark.ru/chtab/service-worker.js` - должен открываться
- `https://deepark.ru/chtab/chords-analyzer.js` - должен открываться

### 2. Проверьте в Chrome DevTools (на компьютере):
1. Откройте `chrome://inspect` в Chrome на компьютере
2. Подключите Android устройство через USB
3. Откройте сайт на Android
4. В DevTools перейдите в Application > Manifest - проверьте, что манифест загружен
5. В Application > Service Workers - проверьте, что Service Worker зарегистрирован

### 3. Проверьте консоль браузера:
- Откройте консоль на Android (через Chrome Remote Debugging)
- Должны быть сообщения: "Service Worker зарегистрирован" и "PWA готово к установке"

### 4. Ручная установка:
Если автоматический баннер не появляется, можно установить вручную:
1. Откройте меню Chrome (три точки)
2. Выберите "Добавить на главный экран" или "Установить приложение"

## Возможные проблемы:

1. **Нет HTTPS** - Chrome требует HTTPS для PWA (кроме localhost)
2. **Нет иконок** - создайте icon-192.png и icon-512.png
3. **Неправильные пути** - проверьте, что все пути в manifest.json и service-worker.js правильные
4. **Service Worker не регистрируется** - проверьте консоль на ошибки
5. **Кэш браузера** - очистите кэш и перезагрузите страницу

## Создание иконок:

Если иконок нет, можно создать простые:
- icon-192.png: 192x192 пикселей
- icon-512.png: 512x512 пикселей

Можно использовать любой графический редактор или онлайн генератор иконок.
