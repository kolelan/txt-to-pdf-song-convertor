# PWA - Progressive Web App

Приложение настроено как PWA (Progressive Web App) и может быть установлено на устройство.

## Установка

### На Android (Chrome):
1. Откройте приложение в браузере Chrome
2. Нажмите на меню (три точки)
3. Выберите "Добавить на главный экран" или "Установить приложение"
4. Подтвердите установку

### На iOS (Safari):
1. Откройте приложение в Safari
2. Нажмите кнопку "Поделиться"
3. Выберите "На экран «Домой»"
4. Подтвердите добавление

### На Desktop (Chrome/Edge):
1. Откройте приложение в браузере
2. Нажмите на иконку установки в адресной строке
3. Подтвердите установку

## Иконки

Для полной функциональности PWA нужны иконки:
- `icon-192.png` (192x192 пикселей)
- `icon-512.png` (512x512 пикселей)

Вы можете создать иконки или использовать временные заглушки.

## Функции PWA

- ✅ Работает офлайн (после первой загрузки)
- ✅ Можно установить на главный экран
- ✅ Работает как нативное приложение
- ✅ Быстрая загрузка благодаря кэшированию
